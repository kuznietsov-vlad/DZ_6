## Запуск програми

Для запуску програми необхідно відкрити термінал у каталозі з файлами main.py та run.py.
Спочатку виконується наповнення бази даних тестовими даними командою:

python main.py

Після цього виконуються SQL-запити з окремих .sql-файлів командою:

python run.py

Усі SQL-запити збережені в окремих файлах та можуть бути виконані як через термінал PostgreSQL, так і через cursor.execute() у Python.
